# Netflix_frontend

Introduction
This project is a simple Netflix frontend clone created using only HTML and CSS. It aims to showcase the layout and design of the Netflix user interface. This is a static representation and does not include backend functionality or streaming capabilities.

Features
Responsive design for various screen sizes.
Basic layout resembling the Netflix interface.
Thumbnail grid showcasing movie or TV show posters.
Navigation bar with a simplified menu.

Structure
The project structure is straightforward:

index.html: The main HTML file containing the structure of the webpage.
styles.css: The CSS file responsible for styling the HTML elements.
img/: A directory containing sample images for movie or TV show posters.
